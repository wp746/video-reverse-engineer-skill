## 模板使用规则

### 必填字段

| 字段 | 是否必填 | 说明 |
|------|---------|------|
| SEG 编号 | ✓ | 格式 `SEG-NN` |
| 段名 | ✓ | 2-6字概括核心事件 |
| 时间范围 | ✓ | 精确到 0.5s |
| 时长 | ✓ | 自动计算 |
| 参考图 | ✓ | 列出本段用到的所有参考资产 |
| 衔接方案 | ✓ | A(首帧截取) 或 B(首帧描述) |
| 🎥 运镜设计表 | ✓ | v2.0新增必填 |
| 提示词正文 | ✓ | 完整英文提示词 |
| ✅ 运镜自检 | ✓ | v2.0新增必填，至少通过6/9项 |
| ✅ 内容自检 | ✓ | 至少通过4/5项 |

### 运镜标记规范

- **★ 主运镜**：每段仅限 1 个，是本段的核心运镜叙事手段
- **☆ 辅运镜**：每段 0-3 个，配合主运镜的辅助技巧
- **编号**：引用「38种运镜分类总表」中的统一编号（01-36）
- **速度参数**：
  - 极慢：< 0.3 cm/s
  - 慢推：0.3-0.8 cm/s
  - 正常：1-3 cm/s 或匹配主体速度
  - 快速：> 5 cm/s
  - 急推/甩镜：< 0.3s 完成
- **焦段标注**：必须写 mm 数值或等效焦段

### 衔接方案定义

| 方案 | 定义 | 适用场景 |
|------|------|---------|
| **方案A — 首帧截取法** | 从前一段输出视频中截取最后一帧作为本段的首帧输入 | 有实际视频输出时 |
| **方案B — 首帧描述法** | 用文字精确描述"从前一段末帧过渡到本段首帧"的画面 | 尚无视频、纯提示词阶段 |

### 时间精度要求

- 开场/收束段：秒级即可（情绪铺垫不需要帧级精度）
- 战斗/动作段：0.5s 精度（每个打击点需要时间锚定）
- 转折/揭示段：帧级理想（但 Seedance 当前不支持帧级，用 0.5s 近似）
