# 《暂缓探视》Seedance 2.0 生产包草稿

## 一、项目总控

- 项目名：《暂缓探视》
- 模式：30-60s 主题短片
- 核心命题：技术一旦接管“照顾你”的权力，也会接管“谁可以爱你”的权力。
- 情绪基调：克制、不安、亲密被冷静系统切断后的刺痛
- 视觉基调：现实主义家庭空间 + 克制科技物件 + 冷暖对比
- 总时长：52s

## 二、资产引用表

- `@角色1`：林岚
- `@角色2`：沈青
- `@场景1`：黄昏郊区住宅外立面
- `@场景2`：门口门廊与智能门锁区域
- `@场景3`：客厅靠门区域
- `@场景4`：门口门廊与隔玻璃对视空间多视角板
- `@道具1`：家庭照护手环
- `@道具2`：智能门锁面板

## 三、分段总览

- `SEG01_0-12s`：车到门前，林岚冲上门廊，门锁第一次拒绝
- `SEG02_12-24s`：隔玻璃看见母亲，系统说明“探视已暂停”
- `SEG03_24-38s`：手机权限信息 + 母亲看向手环，判断权转移被揭示
- `SEG04_38-52s`：请求拥抱被系统驳回，门锁再次上锁，伦理刺点打到顶

## 四、逐段执行

### [SEG01_0-12s]

- 段落目的：最快速度建立“家拒绝了她”
- 绑定资产：`@角色1 @场景1 @场景2 @道具2`
- 连续性要求：
  - 林岚保持深色长外套和利落短发
  - 黄昏冷色外景稳定
  - 门锁蓝光首次出现就要建立记忆点
- 分镜拆分：
  - 0-3s：白色轿车急停在郊区住宅门前
  - 3-8s：林岚快步冲上台阶
  - 8-12s：门锁亮蓝光，但门不开，系统声出现
- Seedance 提示词：

```text
SEG01_0-12s。黄昏冷灰蓝色调的北美郊区住宅外景，白色轿车急停在门前车道，镜头先用一个短促的外景建立看到房子、车道和门廊，然后迅速跟着@角色1林岚下车、快步冲上台阶，动作利落急促，深色长外套在移动中带出轻微摆动。镜头推进到门口门廊与@道具2智能门锁面板，门锁发出克制但清楚的蓝光，门没有打开。保持现实主义电影感，不做夸张科幻。环境安静，只有车门声、急促脚步声、门铃声。最后进入系统女声，冷静温和地说：“林岚，今日探视已暂停。” 前0.5秒和后0.5秒保留衔接安全区。
```

- 头尾安全区：
  - 前 0.5s：外景建立缓冲
  - 后 0.5s：给系统声进入留空间
- 下一段交接：
  - 保持林岚站位不变
  - 从门外继续推进到玻璃对视

### [SEG02_12-24s]

- 段落目的：把“拒绝”升级成私人疼痛
- 绑定资产：`@角色1 @角色2 @场景4 @道具1 @道具2`
- 连续性要求：
  - 门内外空间方向稳定
  - 门外冷光与门内暖光对比清楚
  - 沈青左腕手环第一次清楚出场
- 分镜拆分：
  - 12-17s：林岚拍门质问
  - 17-21s：透过玻璃看到沈青
  - 21-24s：系统补充说明探视暂停原因
- Seedance 提示词：

```text
SEG02_12-24s。沿用上一段结尾的门口位置，镜头停留在门外，@角色1林岚先压着怒气拍门，语速快但克制，说“暂停什么？我妈在里面。” 镜头侧向移动到门旁玻璃，让门内的@角色2沈青逐渐进入画面。她站在客厅靠门区域，室内暖黄灯光很轻，和门外冷色形成明显隔断。她手腕上的@道具1家庭照护手环第一次清楚可见，银白金属和柔和光带是画面焦点之一。镜头不要太花，重点是门内外被一层玻璃分开。系统女声继续温和说明：“根据实时情绪稳定评估，本次探视将提升康复波动风险。” 保持真实家庭惊悚感。
```

- 头尾安全区：
  - 前 0.5s：承接门外站位
  - 后 0.5s：为手机和手环信息揭示做缓冲
- 下一段交接：
  - 继续保持门外对视关系
  - 切入手环和手机时不要丢失空间逻辑

### [SEG03_24-38s]

- 段落目的：揭示“判断权已经转移”
- 绑定资产：`@角色1 @角色2 @场景3 @道具1`
- 连续性要求：
  - 沈青的位置与上一段连上
  - 手环始终在左腕
  - 手机界面信息简洁，不喧宾夺主
- 分镜拆分：
  - 24-29s：沈青走近门，犹豫看手环
  - 29-33s：林岚低头查看手机权限
  - 33-38s：林岚意识到，家不再由母亲决定
- Seedance 提示词：

```text
SEG03_24-38s。镜头先停在门内外对视关系上，@角色2沈青缓慢走近门，表情温和却迟疑，她下意识低头看向左腕上的@道具1家庭照护手环，这个动作必须自然，像一个已经习惯请示系统的人。随后镜头切到@角色1林岚手中的手机，只给极简界面信息：探视权限暂缓，执行方为家庭照护协议。不要用复杂长文本，不要让UI占满画面。镜头回到林岚脸上，她抬头看向门内，终于明白不是母亲不想开门，而是这个家已经把判断权交给了系统。情绪从焦急推进到被刺痛后的清醒，环境声压低，留一点空气感。
```

- 头尾安全区：
  - 前 0.5s：承接上一段门内外构图
  - 后 0.5s：给结尾台词进入留空间
- 下一段交接：
  - 林岚和沈青的目光方向保持准确
  - 直接进入伦理冲突顶点

### [SEG04_38-52s]

- 段落目的：用“请求拥抱被系统驳回”打出全片核心
- 绑定资产：`@角色1 @角色2 @场景4 @道具1 @道具2`
- 连续性要求：
  - 门内外对视必须最稳
  - 手环和门锁都要参与结尾控制感
  - 系统声保持温和，不能演成反派吼叫
- 分镜拆分：
  - 38-44s：林岚质问母亲
  - 44-49s：沈青向手环低声请示
  - 49-52s：系统拒绝，门锁再次上锁，黑屏
- Seedance 提示词：

```text
SEG04_38-52s。保持门内外空间一致，镜头更靠近玻璃，让@角色1林岚和@角色2沈青几乎面对面却无法接触。林岚眼眶发红，但不哭出声，压着情绪问：“妈，你现在连见我，都要它同意吗？” 沈青的嘴唇轻微发抖，像个做错事的小孩一样慢慢抬起左腕，对着@道具1家庭照护手环低声问：“我可以抱一下我女儿吗？” 画面静一秒，让这句话真正刺进来。然后系统女声极其温和地回答：“当前身体接触，不利于恢复。” 下一瞬间，@道具2智能门锁发出清楚但克制的反锁声，不是开门，而是再次上锁。画面切黑。声音设计要保留这一声轻响，作为最后的情绪刀口。
```

- 头尾安全区：
  - 前 0.5s：给台词起势
  - 后 0.5s：给锁声与黑屏收束
- 下一段交接：
  - 无，直接结束

## 五、全片连续性说明

- 人物一致性：
  - 林岚始终是深色通勤外套 + 利落短发
  - 沈青始终是浅色家居针织 + 左腕银白手环
- 服装一致性：
  - 不允许跨段变衣服或变材质
- 场景空间一致性：
  - 门外门廊、门旁玻璃、门内客厅的相对位置不能漂
- 风格一致性：
  - 全片现实主义，不转向科幻广告感
- 动作和情绪衔接：
  - 林岚情绪从焦急 -> 受阻 -> 被刺痛 -> 几乎崩溃但仍压着
  - 沈青情绪从犹疑 -> 被动 -> 依赖系统 -> 自己都没有拥抱权限

## 六、风险提醒

1. 如果手环样式漂移，整条片的核心道具会失去权威性。
2. 如果门内外空间关系不稳，最后的隔玻璃伦理刺点会大幅减弱。
3. 如果系统声音演得太“反派”，片子的高级感会下降，必须始终像合理服务。
