# 《暂缓探视》Seedance 2.0 生产包执行版

## 一、项目总控

- 项目名：《暂缓探视》
- 模式：30-60s 主题短片
- 核心命题：技术一旦接管“照顾你”的权力，也会接管“谁可以爱你”的权力。
- 情绪基调：克制、礼貌、冷静，但越看越刺痛
- 视觉基调：北美郊区现实空间，黄昏冷光，室内暖光，克制科技蓝光
- 总时长：52s

## 二、资产引用表

- `@角色1`：林岚
- `@角色2`：沈青
- `@场景1`：黄昏郊区住宅外立面
- `@场景2`：门口门廊与智能门锁区域
- `@场景3`：客厅靠门区域
- `@场景4`：门口门廊与隔玻璃对视空间多视角板
- `@道具1`：家庭照护手环
- `@道具2`：智能门锁面板

## 三、分段总览

- `SEG01_0-12s`：急停回家，门锁礼貌拒绝
- `SEG02_12-24s`：隔玻璃看见母亲，系统说明暂停探视
- `SEG03_24-38s`：手环 + 权限信息，揭示家中判断权转移
- `SEG04_38-52s`：请求拥抱被驳回，门锁再次上锁

## 四、逐段执行

### SEG01_0-12s

- 段落目的：迅速建立“家拒绝了她”
- 绑定资产：`@角色1 @场景1 @场景2 @道具2`
- 连续性要求：
  - 林岚服装、发型稳定
  - 黄昏冷色稳定
  - 门锁蓝光首次出现就要建立系统记忆点
- 分镜拆分：
  - 0-3s：白色轿车急停在郊区住宅门前
  - 3-8s：林岚快步冲上门廊
  - 8-12s：门锁亮蓝光但门不开，系统声出现
- Seedance 提示词：

```text
SEG01_0-12s。现实主义电影感，黄昏冷灰蓝色调的北美郊区住宅外景。镜头先短暂建立住宅外立面和白色轿车停靠，然后迅速跟住@角色1林岚下车、快步冲上门廊。她穿深色长款通勤外套和高领内搭，动作急、压着情绪。镜头推进到@场景2门口门廊与@道具2智能门锁，门锁发出克制清晰的蓝光，但门没有打开。保持真实家庭空间，不做夸张科幻。环境声只有车门声、脚步声、门铃声。最后让系统女声温和进入：“林岚，今日探视已暂停。” 前0.5秒和后0.5秒保留衔接安全区。
```

### SEG02_12-24s

- 段落目的：把技术性拒绝升级成亲密关系疼痛
- 绑定资产：`@角色1 @角色2 @场景4 @道具1 @道具2`
- 连续性要求：
  - 门内外空间方向不能漂
  - 手环首次出场必须清楚
  - 门外冷光与门内暖光形成稳定隔断
- 分镜拆分：
  - 12-17s：林岚拍门质问
  - 17-21s：透过玻璃看见沈青和手环
  - 21-24s：系统说明暂停原因
- Seedance 提示词：

```text
SEG02_12-24s。承接上一段的门口位置，镜头停留在@场景4的门内外对视空间。@角色1林岚先压着怒气拍门，语速快但不失控，说“暂停什么？我妈在里面。” 镜头微移到门旁玻璃，让门内的@角色2沈青逐渐进入画面。她站在门内客厅靠门区域，左腕佩戴的@道具1家庭照护手环清晰可见，室内暖黄光和门外冷光形成明显隔断。系统女声继续温和说明：“根据实时情绪稳定评估，本次探视将提升康复波动风险。” 重点是门内外被一层玻璃礼貌而残酷地隔开。
```

### SEG03_24-38s

- 段落目的：让观众意识到，家中的判断权已经转移
- 绑定资产：`@角色1 @角色2 @场景3 @道具1`
- 连续性要求：
  - 手环始终在左腕
  - 手机界面只给最短信息，不抢戏
  - 情绪从急躁转成刺痛后的清醒
- 分镜拆分：
  - 24-29s：沈青走近门，低头看手环
  - 29-33s：林岚看到手机权限“暂缓”
  - 33-38s：林岚抬头，理解真正发生了什么
- Seedance 提示词：

```text
SEG03_24-38s。镜头先保持门内外关系，@角色2沈青缓慢走近门，神情温和却迟疑，她低头看向左腕上的@道具1家庭照护手环，这个动作必须自然，像她已经习惯向系统确认。随后短切到@角色1林岚手中的手机，只展示极简信息：探视权限暂缓，执行方为家庭照护协议。不要长篇界面文字。镜头回到林岚脸上，她抬头看门内，终于明白不是母亲不想开门，而是这个家已经把判断权交给了系统。保持克制表演和现实空间逻辑。
```

### SEG04_38-52s

- 段落目的：把伦理刺点打到顶
- 绑定资产：`@角色1 @角色2 @场景4 @道具1 @道具2`
- 连续性要求：
  - 母女对视关系最稳
  - 系统声不能像反派，而要像合理服务
  - 最后一声上锁是全片刀口
- 分镜拆分：
  - 38-44s：林岚质问母亲
  - 44-49s：沈青抬腕请示能否拥抱女儿
  - 49-52s：系统拒绝，门锁再次上锁，黑屏
- Seedance 提示词：

```text
SEG04_38-52s。继续保持@场景4门内外对视空间，镜头更靠近玻璃，让@角色1林岚和@角色2沈青几乎面对面却无法接触。林岚眼眶发红，但声音压着，问：“妈，你现在连见我，都要它同意吗？” 沈青的嘴唇轻微发抖，像个做错事的小孩一样慢慢抬起左腕，对着@道具1家庭照护手环低声问：“我可以抱一下我女儿吗？” 停一秒，让这句话真正刺进来。然后系统女声极其温和地回答：“当前身体接触，不利于恢复。” 下一瞬间，@道具2智能门锁发出清楚但克制的反锁声，不是解锁，而是再次上锁。画面切黑。保留那一下锁声作为最后记忆点。
```

## 五、全片连续性说明

- 人物一致性：
  - 林岚必须始终是短发、深色通勤外套、压着情绪的都市女性
  - 沈青必须始终是浅色居家恢复期服装、左腕银白手环、温和迟疑的状态
- 服装一致性：
  - 四段内不能换衣服、换材质或变主轮廓
- 场景空间一致性：
  - 门外门廊、玻璃侧窗、门内客厅靠门区域必须始终能互相对上
- 风格一致性：
  - 全片必须是现实主义家庭惊悚，不是高概念科幻广告
- 动作情绪衔接：
  - 林岚：急迫 -> 被拒绝 -> 被刺痛 -> 几乎崩裂
  - 沈青：犹疑 -> 被动 -> 依赖系统 -> 连拥抱都需要请示

## 六、风险提醒

1. 如果门内外空间逻辑不稳，隔玻璃对视的所有力量都会掉下来。
2. 如果系统声音演得太邪恶，这条片就会掉到低级反派戏，而不是“合理服务的恐怖”。
3. 如果手机界面文字过多，会稀释门锁和手环的影像力量。
