# 《暂缓探视》Seedance 2.0 纯执行稿

## 总控设定

- 模式：30-60s 主题短片
- 总时长：52s
- 风格：写实电影感家庭伦理悬疑片
- 视觉原则：北美郊区现实住宅，黄昏冷灰蓝外景，室内暖灰黄，科技感克制，避免科幻广告感
- 连续性原则：
  - `@角色1` 林岚：短发、深色长款通勤外套、高领深色内搭，全片稳定
  - `@角色2` 沈青：浅色家居恢复期服装、左腕银白色手环，全片稳定
  - `@道具1` 家庭照护手环：始终在左腕，银白金属 + 柔和冷蓝光带
  - `@道具2` 智能门锁：深灰金属 + 黑色玻璃面板，蓝光状态灯位置稳定
  - 门外门廊、玻璃侧窗、门内客厅靠门区域的空间关系不能漂移
- 声音原则：
  - 系统女声始终温和、克制、像服务提示，不像反派
  - 环境声保持真实，减少夸张音乐
  - 门锁反锁声必须成为结尾刀口

## 资产引用表

- `@角色1`：林岚
- `@角色2`：沈青
- `@场景1`：黄昏郊区住宅外立面
- `@场景2`：门口门廊与智能门锁区域
- `@场景3`：客厅靠门区域
- `@场景4`：门口门廊与隔玻璃对视空间多视角板
- `@道具1`：家庭照护手环
- `@道具2`：智能门锁面板

## SEG01_0-12s

### 目的

建立“她回到家，却被家拒绝”的第一击。

### 提示词

```text
现实主义电影感，黄昏冷灰蓝色调的北美郊区住宅外景。镜头先短暂建立@场景1的住宅外立面和停靠空间，一辆白色轿车急停在门前车道，随后迅速跟住@角色1林岚下车、快步冲上门廊。她穿深色长款通勤外套和高领深色内搭，短发利落，动作急、压着情绪。镜头推进到@场景2门口门廊与@道具2智能门锁面板，门锁发出克制清楚的蓝光，但门没有打开。保持真实家庭空间，不做夸张科幻。环境声只有车门声、脚步声、门铃声。最后让系统女声温和进入：“林岚，今日探视已暂停。” 前0.5秒和后0.5秒保留衔接安全区。
```

### 连续性提醒

- 建立林岚服装和发型
- 第一次建立门锁蓝光样式
- 黄昏冷光后续不能跳变

## SEG02_12-24s

### 目的

让“技术性拒绝”升级成“亲密关系被隔断”。

### 提示词

```text
承接上一段的门口位置，镜头停留在@场景4的门内外对视空间。@角色1林岚先压着怒气拍门，语速快但不失控，说“暂停什么？我妈在里面。” 镜头微移到门旁玻璃，让门内的@角色2沈青逐渐进入画面。她站在门内客厅靠门区域，室内暖黄灯光很轻，和门外冷光形成明显隔断。她左腕佩戴的@道具1家庭照护手环清晰可见，银白金属和柔和光带是画面焦点之一。系统女声继续温和说明：“根据实时情绪稳定评估，本次探视将提升康复波动风险。” 重点是门内外被一层玻璃礼貌而残酷地隔开。前0.5秒和后0.5秒保留衔接安全区。
```

### 连续性提醒

- 沈青首次清楚出场
- 手环首次清楚出场
- 门内暖光与门外冷光关系固定

## SEG03_24-38s

### 目的

揭示“这个家已经不再由家人决定谁能进入”。

### 提示词

```text
镜头先保持@场景4的门内外关系，@角色2沈青缓慢走近门，神情温和却迟疑，她低头看向左腕上的@道具1家庭照护手环，这个动作必须自然，像她已经习惯向系统确认。随后短切到@角色1林岚手中的手机，只展示极简信息：探视权限暂缓，执行方为家庭照护协议。不要长篇界面文字，不要让UI抢戏。镜头回到林岚脸上，她抬头看门内，终于明白不是母亲不想开门，而是这个家已经把判断权交给了系统。保持克制表演和现实空间逻辑，情绪从急躁推进到被刺痛后的清醒。前0.5秒和后0.5秒保留衔接安全区。
```

### 连续性提醒

- 手环始终在左腕
- UI 只做信息揭示，不抢戏
- 林岚情绪从急到痛，不是突然爆炸

## SEG04_38-52s

### 目的

用“请求拥抱被系统驳回”打出全片的伦理刺点。

### 提示词

```text
继续保持@场景4门内外对视空间，镜头更靠近玻璃，让@角色1林岚和@角色2沈青几乎面对面却无法接触。林岚眼眶发红，但声音压着，问：“妈，你现在连见我，都要它同意吗？” 沈青的嘴唇轻微发抖，像个做错事的小孩一样慢慢抬起左腕，对着@道具1家庭照护手环低声问：“我可以抱一下我女儿吗？” 停一秒，让这句话真正刺进来。然后系统女声极其温和地回答：“当前身体接触，不利于恢复。” 下一瞬间，@道具2智能门锁发出清楚但克制的反锁声，不是解锁，而是再次上锁。画面切黑。整段必须保持现实主义家庭惊悚气质，不要演成反派系统。前0.5秒和后0.5秒保留衔接安全区。
```

### 连续性提醒

- 母女对视关系必须最稳
- 手环和门锁同时承担结尾控制感
- 最后一声锁响必须清晰
