# 好莱坞原型重构系统 (Hollywood Prototype Reconstruction System)

一个面向 AIGC 短片与创意视频创作者的**工业级前置创意重构与资产/视频提示词编译系统**。

本系统支持导入视频文件、链接或截图，自动触发 **10 维全维度原型拆解**（剧情、角色、风格、道具、台词、构图、运镜、特效、音效、叙事节奏），完美支持风格转译（如动画转写实）与主题结构迁移，并自动为后端 **image2 (Google Banana 2) + seedance2.0** 生成无缝衔接的资产 JSON 与提示词包。

## 🎯 核心解决的问题

原型重构系统旨在解决 AIGC 视频工业化生产中的四大痛点：

1. **浅层模仿（照抄）**：仅复制原片字面台词和场景，导致作品缺乏灵魂或涉嫌抄袭。本系统通过**“剥离皮囊，保留骨架”**的原型方法，帮助您重构同等情感压强的新作。
2. **画风转译失真**：从 2D 动漫或 3D 动画转为写实电影时，画面质感发飘、缺少真实感。本系统建立“风格退耦与写实重建”逻辑，强制锁死材质细部。
3. **资产与视频提示词发飘**：人物面部漂移、服装跨片段不一致。本系统使用前置 **image2 (Banana 2) JSON 锁定静态一致性**，再通过 **`@角色` 绑定 seedance2.0 驱动动态运动**。
4. **视频生成 Bug 频发**：特效没有依附、运镜没有物理参数、剪辑点产生畸变。本系统内置 **8 条工业级提示词核心规律**（安全区、变化链等）确保一次性高成功率出片。

## ⚙️ 核心工作流 (Workflow)

系统严格按照好莱坞工业化制片流程设计，不支持跳步推进：

```text
视频输入 -> 10维原型拆解 -> 风格与主题迁移 -> 用户选定方向 -> 重构脚本 -> 顺序化专家团审查 -> 制片主任拆资产 -> image2 JSON资产包 -> 分镜时间轴 -> seedance2.0 提示词包 -> 交付
```

## 💎 原型 10 维拆解标准

系统会对输入的参考片进行以下 10 个维度的极限拆解：

1. **逐剧情 (Plot)**：拆解起承转合结构与分镜头的戏剧任务。
2. **逐角色 (Character)**：识别角色功能原型、人际关系与视觉识别锚点。
3. **逐风格 (Style)**：解析色彩冷暖温值、光影明暗法则、材质纹理与画风介质。
4. **逐道具 (Prop)**：提炼叙事互动型道具与情感象征型道具。
5. **逐台词 (Dialogue)**：分析台词旁白风格、哲理锚定金句与声画配比关系。
6. **逐构图 (Composition)**：提取三分线、绝对对称、广角透视与景深原理。
7. **逐运镜 (Camera Motion)**：还原机位轨迹、镜头焦段、运动阻尼与抖动系数。
8. **逐特效 (VFX)**：界定粒子、流体与大气特效的空间依附与材质折射反射。
9. **逐音效 (Sound & Ambience)**：勾勒声场回音空间、动作同步音效与低频氛围。
10. **逐叙事节奏 (Pacing)**：统计平均镜头时长、戏剧张力压缩与释放时间点。

## 📜 8 条工业级提示词规律

为保证后端 **seedance2.0** 高效出片，本技能强制推行 8 大提示词编写铁律：

1. **镜头职责清晰**：按秒数切块，明确分配画面转折任务。
2. **先锁连续性，再写变化**：首句声明 `@角色`、`@场景` 外观，随后描写动态。
3. **运镜物理化**：具体到机位高度、移动轨迹、平稳度阻尼及终点落幅。
4. **特效依附空间与材质**：特效光流必须沿物理材质漫延，产生真实漫反射。
5. **情绪靠表演与光影**：用呼吸频率、视线方向、斜侧光影传达情绪，不用空洞形容词。
6. **控制变化链**：严密规划“主体动作 $\rightarrow$ 镜头跟随 $\rightarrow$ 环境反馈 $\rightarrow$ 稳定收尾”的因果回路。
7. **预留首尾安全区**：每段提示词首尾预留 1.5 秒锁定稳定微动，防止拼接时镜头扭曲。
8. **动静分工原则**：静态外观交给 image2 资产板锁死，视频提示词仅专注于**“变化的增量”**。
