# 运镜设计指南 (Cinematography Design Guide)

> **版本**: v1.0 — 基于38种专业运镜技巧分类体系
> **来源**: 运镜技巧分类总表（六大类38种）
> **用途**: Seedance 2.0 提示词阶段的运镜决策工具

---

## 核心哲学

> *"好的导演也会运用运镜技巧去表达情绪，讲故事，推进剧情，制造节奏。"*

在 Seedance 2.0 的图生视频流程中，**运镜提示词是唯一能控制"怎么看"的手段**。运镜不是"附加描述"而是**表达性的电影语言**——每种运镜的选择都服务于该段落的情绪目标和叙事功能。

## 运镜 × SEG 匹配决策流程

### Step 1: 分析该段的情绪任务

问自己：
- 这一段的情绪是什么？（安静/压抑/爆发/释放/回归）
- 观众应该**感觉**到什么？
- 这一段在全片中承担什么**叙事功能**？

### Step 2: 从六类中选择主运镜

按情绪→运镜的直接映射：

| 如果情绪是... | 主运镜候选来自... |
|-------------|------------------|
| 安静/蓄力/内心 | 类①: 01慢推, 06光学变焦 |
| 压迫/威胁逼近 | 类②: 10下摇 + 类⑥: 35长焦压缩, 32低角跟拍 |
| 爆发/暴力释放 | 组合: 类①急推 + 类②升降 + 类④手持/甩镜 + 类⑥跟踪 |
| 流动/自由 | 类⑥: 33侧向跟踪 + 类②抬升/弧形 |
| 不安/窒息 | 类④: 23手持(抖动强度变化) + 16荷兰角(可持续/临时) |
| 量级跃迁/质变 | 类④: 04微距 + 类①: 03畸变 + 类④: 19遮挡揭示 |
| 全能展示/高光 | 类③: 13半圆环绕 + 类⑤: 28上帝视角 |
| 混乱/失控 | 类③: 14快速360 + 类④: 16荷兰角(持续) + 类⑥: 32低角压迫 |
| 终极一击/Y轴极致 | 类②: 25抬升→类①: 急推→类④: 微距 → Y轴完整叙事 |
| 回归/闭环 | 类①: 01慢推(镜像) + 类⑥: 32背影跟拍 + 类②: 09向上摇 |

### Step 3: 选择辅助运镜（1-2个）

辅助运镜的作用：
- **节奏标点**: 在关键时间点用不同运镜"打拍子"（如每次锤击时 Crash Zoom）
- **维度补充**: 主运镜只管一个维度，辅助运镜补充另一个维度（如主运镜是水平跟踪，辅运镜是垂直 Crane Up）
- **情绪对比**: 用短暂的相反运镜制造反差（如稳定环绕中突然插入一个 Dutch Angle）

### Step 4: 写入 🎥 CINEMATOGRAPHY DESIGN 区块

每个 Seedance 提示词必须包含独立的运镜设计区块：

```
─── CAMERA MOVEMENT (CINEMATOGRAPHY DESIGN) ───
PRIMARY TECHNIQUE: [名称] ([编号] from [类别]).
[参数: 速度/角度/轨迹/加速度曲线]

SECONDARY: [名称] ([编号]) — [何时触发/为什么]

WHY THIS CHOICE: [一两句话说明这个运镜如何服务于本段的情绪目标]

CAMERA TRAJECTORY: [具体路径描述]
```

### Step 5: 写入 ✅ 运镜自检清单

每个段必须有运镜专属自检：
- [ ] 主运镜技术名称正确且来自38种分类表
- [ ] 运镜参数具体（速度/角度/距离/持续时间）
- [ ] 辅助运镜的触发时机明确
- [ ] 运镜选择与情绪目标的映射逻辑成立
- [ ] 运镜描述已嵌入英文提示词的 `─── CAMERA MOVEMENT ──` 区块
- [ ] 若有多运镜组合，衔接/过渡合理

## 高级设计模式

### 模式 A: 运镜呼吸（Breathing Cinematography）

**适用**: 跑酷、多阶段动作段落
**做法**: 运镜强度随动作密度起伏——动作密集时运镜也复杂/快，动作舒缓时运镜也稳/慢
**示例**: SEG-04 跑酷战 —— 跟踪(稳)→抬升(升空)→长焦压缩(聚焦水面)→荷兰角(落地不稳)→回 tracking(恢复)

### 模式 B: 运镜过山车（Rollercoaster Cinematography）

**适用**: BOSS战、多回合段落
**做法**: 每个回合用不同主导运镜，运镜情绪值随回合递增
**示例**: SEG-08 BOSS战 —— Round1 稳定环绕(战术观察) → Round2 弧形+间歇Dutch角(游击) → Round3 Fast360+持续Dutch角+低角压迫(绝望)

### 模式 C: 运镜镜像对称（Mirror Symmetry）

**适用**: 收束 vs 开场 形成闭环
**做法**: 收束段的主运镜 = 开场段的主运镜，但所有参数都有意微调（更慢/更柔/焦点不同）
**示例**: SEG-10 Slow Dolly In = SEG-01 Slow Dolly In（但速度 0.4 vs 0.5 cm/s，焦点从橙色威胁→暖光微笑）
**对称维度**（可检查7维度）: 运镜类型/速度/方向/焦点目标/光线暗示/表情同步/后续衔接

### 模式 D: Y轴叙事（Vertical Axis Storytelling）

**适用**: 终极技、量级最大段落
**做法**: 运镜完全工作在Y轴上—— Crane Up(地→天) → 顶点滞空(Tilt Down看下方) → Free Fall descent → Rush In+Micro impact → Pull back to ground
**示例**: SEG-09 天坠锤 —— 完整的垂直叙事弧线

## 踩坑经验（从实战中积累）

- **运镜不要贪多**: 每段 1 个主运镜 + 1-2 个辅助运镜足够。超过 3 种会让 Seedance 不知道该听谁的
- **运镜描述要给参数**: 不要只写 "dynamic camera"，要给速度/距离/角度/持续时间
- **运镜变化要有理由**: 每次运镜切换都应该对应一个**情绪变化或动作节拍**
- **Seedance 的运镜限制**: 极端复杂的运镜组合（比如同时做 Orbit + Handheld + Dolly Zoom）可能无法完美执行，要分主次
- **运镜要与动作同步**: 运镜速度≈动作速度 是最自然的，不需要刻意制造反差除非有特殊表达目的
- **首尾运镜安全区**: 段落前0.5s和后0.5s避免剧烈运镜变化（方便衔接）
