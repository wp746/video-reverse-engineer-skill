# Script Review Order

Use this file to run industrial review after the user chooses a direction and a script draft is written.

## Core principle

The review is sequential, not parallel.

The script moves forward only when each stage clears its gate.

## Review sequence

### Gate 1: Chief Director Review

Primary focus:

- proposition clarity
- expressive unity
- emotional force
- whether the piece has a reason to exist

Fail fast if:

- theme is vague
- expression is scattered
- the piece feels visually dressed up but conceptually empty

Suggested weight:

- 20 points

Minimum pass suggestion:

- 18/20

### Gate 2: Screenwriter Review

Primary focus:

- structure
- character motivation
- emotional progression
- dialogue / voiceover effectiveness

Suggested weight:

- 20 points

Minimum pass suggestion:

- 18/20

### Gate 3: Cinematography / Storyboard Review

Primary focus:

- whether the script can actually become shots
- shot density and rhythm
- visual staging feasibility

Suggested weight:

- 15 points

Minimum pass suggestion:

- 13/15

### Gate 4: Art / Style Review

Primary focus:

- visual identity
- image system
- whether style supports theme rather than decorating it

Suggested weight:

- 10 points

### Gate 5: Costume / Props Review

Primary focus:

- whether characters, props, and material details support story world
- whether later asset generation can stay consistent

Suggested weight:

- 10 points

### Gate 6: Post / Sound Review

Primary focus:

- sound design potential
- pacing closure
- whether the script already suggests a believable finished-piece rhythm

Suggested weight:

- 10 points

### Gate 7: AI Feasibility Review

Primary focus:

- whether the script is realistically producible in AI video workflow
- whether complexity is controllable
- whether later Seedance prompts can remain stable

Suggested weight:

- 15 points

Minimum pass suggestion:

- 13/15

## Pass rule

The script passes only if:

- total score >= 90
- all key gates meet their minimum threshold
- no key reviewer issues a structural veto

## Structural veto examples

- “Theme not established”
- “No producible shot logic”
- “AI execution risk is unacceptable”

## Review output format

Each reviewer should output:

- score
- pass / fail
- core diagnosis
- top problems
- revision instructions

At the end, produce:

- total score
- pass / fail
- blocking issues
- next-step instruction
