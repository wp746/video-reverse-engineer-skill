# Analysis Framework

Use this file when performing the reverse-engineering phase.

## Purpose

Turn a reference video into an analyzable structure that can later support:

- derivative ideation
- script writing
- asset extraction
- storyboard design
- Seedance production

## Output layers

### 1. Surface record

Capture:

- duration
- number of major shots
- timestamps
- what objectively happens in each shot
- dialogue / subtitles / sound cues

### 2. Creative logic

Ask:

- what is the real proposition of the piece
- what emotional after-effect is it aiming for
- why does it feel “deep” instead of just “beautiful”

### 3. Visual logic

Analyze:

- palette
- lighting
- framing
- movement tendency
- recurring imagery

### 4. Narrative logic

Analyze:

- how information is released
- where escalation happens
- where reversal or compression happens
- how the ending lands

### 5. Shot logic

For each major shot or shot-group, identify:

- shot purpose
- shot size
- angle
- movement
- action
- emotional task
- relation to previous and next shots

### 6. Language logic

Analyze:

- dialogue function
- voiceover function
- whether words reinforce or counterpoint the image
- which line acts as the viewpoint anchor

### 7. Sound logic

Analyze:

- music function
- ambience function
- action sound function
- where sound is lifting emotion versus supplying information

### 8. Migration logic

Conclude:

- what to preserve
- what to replace
- what structure can be reused without copying surface text

## Recommended granularity by mode

### 15s emotion short

- shot-by-shot analysis
- key-second emphasis
- emotional curve must be explicit

### 30-60s theme short

- shot-by-shot analysis
- key-second emphasis on turning points
- connect theme to image and sound

### 1-3min narrative short

- scene / shot-block analysis
- detailed analysis on crucial transitions and climax points
- avoid empty second-by-second verbosity
